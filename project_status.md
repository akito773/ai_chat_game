## 🚀 使用方法

### 1. 即座に動作確認
1. `index.html` をブラウザで直接開く
2. さくらちゃんとチャット開始！（ローカル応答モード）

### 2. 本格的なAI連携
1. `api/config.php` でAPIキーを設定
2. `js/config.js` で `useAPI: true` に変更
3. `start_server.bat` でサーバー起動
4. http://localhost:8000 でアクセス

### 3. トラブルシューティング
- `test.html` で動作確認
- ブラウザの開発者ツールでエラーチェック
